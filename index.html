SmartReply — WhatsApp Chatbot

Problem

A local clothing store in Jaipur was losing customers because nobody replied to WhatsApp messages after 10 PM. Potential buyers would ask "price kya hai?" and get no reply until morning — by then they'd moved on.

Solution

Built a WhatsApp chatbot using Twilio API + n8n that automatically replies to common questions about price, availability, timings, and location — 24/7, instant.

Result

Owner stopped manually replying to 30+ messages/day. Response time went from hours to 3 seconds.

OrderFlow — Google Sheet to WhatsApp

Problem

A homemade food seller tracked all orders in a Google Sheet manually and then had to separately WhatsApp each customer their order confirmation — tedious, error-prone, time-consuming.

Solution

Built an n8n workflow that watches the Google Sheet for new rows. The moment an order is added, it automatically sends a formatted WhatsApp confirmation to the customer with order details and estimated delivery time.

Result

Confirmation time reduced from 30 minutes to instant. Zero missed confirmations.
